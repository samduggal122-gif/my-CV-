
# Samaira's Digital CV

A playful and personal digital CV website built to introduce **Samaira Duggal** — a BCA student exploring full stack web development and experimenting with agent-assisted coding.

This project combines a simple one-page portfolio layout with a charming CSS-animated robot mascot to make a resume site feel more alive, memorable, and fun.

## Preview

Samaira's Digital CV is designed as a single-page website that blends:
- A short personal introduction
- Contact links
- A cute animated robot scene made entirely with HTML and CSS

It is a small but creative front-end project that shows personality along with technical curiosity.

## About the Project

This website serves as a digital CV and personal profile page. Instead of using a plain resume-style layout, it adds a visual identity through a custom robot mascot and a bold color palette.

The project is currently built as a **single HTML file**, with all structure, styling, and animation included together for simplicity. It is planned to be expanded later into separate HTML and CSS files, with support for more languages and additional sections.

## Features

- Single-page digital CV layout
- Semantic content sections for intro and contact details
- CSS-only animated robot mascot
- Soft glow and pulse effects for a playful visual feel
- Strong custom color palette with dark background and bright accents
- Beginner-friendly structure for easy editing
- Ready to host on GitHub Pages

## Tech Stack

- **HTML5**
- **CSS3**
- **Flexbox**
- **CSS animations**
- **GitHub Pages** for deployment

## Design Highlights

### Personal Content Section
The top section introduces Samaira with a short bio, personality-filled text, and contact links.

### Cute Robot Mascot
The robot is the visual centerpiece of the project. It includes:
- A glowing animated background
- Rounded robot head and body
- Screen face with expressive features
- Arms layered with depth using `z-index`
- Reusable animation ideas for future enhancements

### Color Style
The design uses:
- A dark navy background
- Bright cyan robotic highlights
- White and light gray robot tones
- Warm red and pink text accents for headings and labels

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/samduggal122-gif/my-CV-.git
   ```

2. Open the project folder:
   ```bash
   cd my-CV-
   ```

3. Launch `index.html` in your browser.

That’s it — no build tools or installation required.

## After deployment, the site will be live at:

```bash
https://samduggal122-gif.github.io/my-CV-/
```
